# **App Name**: DevOps 3D Portfolio

## Core Features:

- Interactive 3D Environment: Display a dynamic 3D environment using Three.js to showcase DevOps concepts.
- Animated Skill Cards: Showcase skills using animated 3D cards with categorized skills and tool icons.
- Experience Timeline: Present professional experience in a visually appealing timeline format with DevOps tool highlights.
- Contact Form: Allow visitors to send messages directly through the website.

## Style Guidelines:

- Primary color: Neon blue (#39C0ED) to represent technology and reliability.
- Background color: Dark gray (#282A3A) to enhance the neon color contrast and provide a modern feel.
- Accent color: Neon purple (#A6ACCD) to highlight interactive elements and call-to-actions.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-oriented feel; Body font: 'Inter' (sans-serif) for readability.
- Use floating icons (Docker, AWS, Jenkins, Git) for DevOps tools to enhance the 3D environment.
- Use smooth animations with GSAP for parallax scrolling, hover effects, and transitions between sections.
- Responsive design with clear section divisions to provide optimal viewing experience on all devices.
